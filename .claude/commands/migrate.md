マイグレーション（移行）を支援します。

## 対応するマイグレーション

| 種類 | 説明 |
|------|------|
| **パッケージ** | npm→Bun、yarn→Bun |
| **フレームワーク** | CRA→Vite、Express→Hono |
| **言語** | JavaScript→TypeScript |
| **バージョン** | React 17→18、Node 18→20 |

## 実行内容

1. **現状分析**
   - 依存関係の確認
   - 非互換性の検出
   - 移行対象のリストアップ

2. **移行計画**
   - ステップバイステップの手順
   - リスク評価
   - ロールバック方法

3. **移行実行**
   - 自動変換可能な部分を実行
   - 手動対応が必要な箇所をリスト化

## オプション

- : 変更せずに計画のみ
- : バックアップ作成

$ARGUMENTS で移行対象を指定。
